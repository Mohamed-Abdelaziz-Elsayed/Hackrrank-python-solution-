
# Python Division

Given two integers, a and b, perform integer division and float division.

Write a program that reads two integers from standard input, computes:

- the result of integer division (using floor division)
- the result of float division

Print the integer division result on the first line and the float division result on the second line.

## Input Format

A single line containing two space-separated integers, a and b.

## Constraints

- -10000 <= a, b <= 10000
- b != 0

## Output Format

Print the result of the integer division (a // b) on the first line.
Print the result of the float division (a / b) on the second line.

## Sample Input

```py
4
3
```

## Sample Output

```py
1
1.3333333333333333
```

## Explanation

For a = 4 and b = 3:

- Integer division: 4 // 3 = 1
- Float division: 4 / 3 = 1.3333333333333333
