# Print Function

## Problem Description

The `print()` function is used to output values in Python. By default, it adds a newline character at the end of the output.

You need to print the numbers from 1 to N on the same line, without spaces between them.

## Input Format

A single integer N.

## Output Format

Print the numbers from 1 to N on the same line, without any spaces between them.

## Constraints

- 1 ≤ N ≤ 1000

## Example

### Input
```py
3
```

### Output
```
123
```
