# Capitalize

## Task

Given a full name, capitalize the name appropriately. You are asked to ensure that the first and last names of people begin with a capital letter in their passports. For example, `alison heck` should be capitalized correctly as `Alison Heck`.
## Example

**Input:**
```py
chris alan
```
**Output:**
```py
Chris Alan
```

## Input Format

The first line contains the full name as a string.

## Constraints
- The string consists of alphanumeric characters and spaces
- In a word, only the first character is capitalized
- Example: `12abc` when capitalized remains `12abc`
## Output Format
Print the capitalized string,S.
## Sample Input
```py
alison heck
```
## Sample Output
```py
Alison Heck
```
